# How does my code run?
## Step 1: Compilation
The code we write is the source code(.java) then this code is send to a compiler that is stored in JDK(java development kit). the compiler converts source code to byte code (.class). this byte code can run in any system as long as it has JRE(java runtime environment). Thats why java is a portabl language. 
This byte code is converted to native code(computer language like binary) by JVM(java virtual machine). 
## Output
sout = print for normal print and println for next line printing.
\n = next line after this.
